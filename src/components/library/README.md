# Component Library

Opgeslagen componenten van 21st.dev en andere bronnen. Deze dienen als bouwblokken bij het maken van klantwebsites.

## Hoe te gebruiken

1. Vind een component die je mooi vindt op 21st.dev, Dribbble, of een referentie-site
2. Zeg: "Sla deze hero op" en geef de code
3. Het component wordt opgeslagen in de juiste map hieronder
4. Bij het bouwen van een klantsite: kies uit deze bibliotheek

## Mappen

- `heroes/` — Hero sectie varianten
- `services/` — Diensten/features secties
- `testimonials/` — Testimonial designs
- `about/` — Over ons / team secties
- `contact/` — Contact formulieren
- `navigation/` — Headers en navigatie
- `footers/` — Footer designs
- `misc/` — Overige componenten

## Naamgeving

`{beschrijving}.tsx` — bv. `elegant-split.tsx`, `minimal-centered.tsx`, `dark-overlay.tsx`
